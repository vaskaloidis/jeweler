Files in this folder are provided for the operation of the dxExporter widget. Since v16.1, this widget is deprecated. Instead, consider using the "export" object of any Data Visualization widget.

The use of the dxExporter widget requires tuning up additional tools. For more information, refer to the following documents.
 
- http://js.devexpress.com/Documentation/Guide/Data_Visualization/Common/Client-Side_Exporting_and_Printing#Exporting_and_Printing_Using_dxExporter - a guide showing how to configure the dxExporter widget and required tools
 
- http://js.devexpress.com/Documentation/ApiReference/Data_Visualization_Widgets/dxExporter/ - dxExporter Reference providing detailed information about the widget options and methods
