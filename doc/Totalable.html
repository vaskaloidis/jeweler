Overview

Adds methods to get total payments, cost, hours, planned-cost and planned-hours

of the tasks below. Available for Projects (all tasks) and Sprints (sprint tasks).

def hours
  if tasks.empty? || tasks.sum(:hours).nil?
    0
  else
    tasks.sum(:hours)
  end
end
